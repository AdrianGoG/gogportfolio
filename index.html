      Interactive map with the following features:

    1. The map will show a default location
    2. Below the map there will be a location search field, with two "search" and "add location" buttons.
    3. If the user enters a location and clicks on "search", the map re-focuses on that location and a location appears on the map.
    4. If the user clicks on "add location", a new map appears on the page that receives this new location
    5. This new map will always keep all the locations added by the user between its visible edges
